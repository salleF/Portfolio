Meus Projetos

Jogo de Ritmo

Um jogo de ritmo desafiador com música envolvente.

Sobre o Projeto

Este jogo foi desenvolvido utilizando a engine Godot com foco em mecânicas de precisão. O maior desafio foi sincronizar perfeitamente as notas musicais com os inputs do jogador.

Tecnologias

  • - Godot Engine
  • - GDScript
  • - Aseprite para Pixel Art

Aprendizados

Aprendi muito sobre gerenciamento de estado em jogos e otimização de assets para garantir uma performance fluida.
